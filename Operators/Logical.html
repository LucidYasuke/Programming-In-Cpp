Logical Operators

Logical operators are used to determine the logic between variables or values

Operator Name Description Example
&& And Returns true if both statements are true x > 5 && x <= 10
|| Or Returns true if one of the statements is true x < 5 || x >= 10
! Not Returns the reverse of the result !(x < 5)

Research DeMorgans Law for a more indepth on the how too use the ! symbol.

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {
                                        // Will print 0 if false, 1 if true
                                        int x = 7;
                                        int y = 32;
                                        int z = 99;

                                        // Check if x is between 1 and 10, but not equal
                                        // If x > 1 is true and x < 10 is true, then it is true
                                        // true && true is true
                                        std::cout << (x > 1 && x < 10) << std::endl; // Output: 1

                                        // Check if y greater than or equal to 50 or divisible by 4
                                        // If 4 divides into y with no remainder then y is a multiple of 4
                                        // false or true is true
                                        std::cout << (y >= 50 || y % 4 == 0) << std::endl; // Output: 1

                                        // If not z greater than y 
                                        // Not true is false
                                        std::cout << (!(z > y)) << std::endl; // Output: 0
                                        return 0;
                                    }
