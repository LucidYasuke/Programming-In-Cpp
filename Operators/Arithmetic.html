Arithmetic Operators

Before we start, operators are symbols in a coding language that has a special tesk. An operator in the real world would be the plus symbol. When we see a plus symbol we know to add. Similarly, a plus symbol in code represents addition. Arithmetic operators are used to perform mathematical operations.

Operator Name Description Example
+ Addition Adds together two values x + y
- Subtraction Subtracts one value from another x - y
* Multiplication Multiplies two values x * y
/ Division Divided one value by another x / y
% Modulus Returns the division remainder x % y
++ Increment Increases the value by 1 x++
-- Decrement Decreases the value by 1 --x

With Increments and Decrements, you can do ++x or x++, however, putting the operator before will cause a pre-increment while the other would be a post-increment. One is done at the begining of the code, while the other is done last. Modulus returns the remainder. So 10 / 3 will give you 3 while 10 % 3 will give you 1. 

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {

                                        int num1 = 10;
                                        int num2 = 3;

                                        std::cout << num1 + num2 << std::endl; // Output: 13
                                        std::cout << num1 - num2 << std::endl; // Output: 7
                                        std::cout << num1 * num2 << std::endl; // Output: 30
                                        std::cout << num1 / num2 << std::endl; // Output: 3
                                        std::cout << num1 % num2 << std::endl; // Output: 1

                                        // Example Of Pre-Increment
                                        std::cout << --num1 << std::endl; // 9
                                        std::cout << num1 << std::endl; // 9
                                        
                                        // Example Of Post-Increment
                                        std::cout << num2++ << std::endl; // 3
                                        std::cout << num2 << std::endl; // 4
                                        return 0;
                                    }
