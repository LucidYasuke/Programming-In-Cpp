Assignment Operators

Assignment operators are used to assignment a value too a variable.

Operator Example Same As
= x = 4 x = 4
+= x += 4 x = x + 4
-= x -= 4 x = x - 4
*= x *= 4 x = x * 4
/= x /= 4 x = x / 4
%= x %= 4 x = x % 4 
                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {

                                        int x = 10;

                                        std::cout << x << std::endl; // Output: 10

                                        x+=4;
                                        std::cout << x << std::endl; // Output: 14

                                        x-=6;
                                        std::cout << x << std::endl; // Output: 8

                                        x*=8;
                                        std::cout << x << std::endl; // Output: 64

                                        x/=32;
                                        std::cout << x << std::endl; // Output: 2

                                        x%=4;
                                        std::cout << x << std::endl; // Output: 2
                                        
                                        // Note : 4 can't go into 2 so the remainder is 2
                                        return 0;
                                    }
