Functions

❮ Previous Next ❯

In programming we constantly repeat code. As we see, onc way we get by this is loops. With loops we can avoid repeating code. But what about the loop itself. What if we want to repeat that code in another file. We don't want to waste time rewriting that code. This is where functions come in. A function can be created to run repeated code. You've been using functions, such as int main(), and vector.size() are examples of functions. One way to tell a function is that they have parenthesis.

Syntax
                                
                                    void foo()
                                    {
                                        // Code block to be executed
                                    }
                                
                            
                                
                                    data_type foo()
                                    {
                                        // Code block to be executed
                                        return data_type;
                                    }
                                
                            
                                
                                    data_type foo(data_type_one parameter_one, data_type_2, parameter_two, ......)
                                    {
                                        // Code block to be executed
                                        return data_type;
                                    }

Above we define a function called foo. If foo was inplemented in a program, it will do everything inside the code block. void is a place holder for a data type. With functions, a singular data of a data_type can be returned to the program. Void is a placeholder, telling the computer that nothing is being returned and everything is staying inside the function. If we wanted to return a data type, we'd have to set the return type at the top. We also have parameters, these are values they may be inputed into the function. Not every function will be empty, some need outside values in order too work.

Example
                                    
                                        #include <iostream>
                                        
                                        // Returns an exponent
                                        int exp(int number, int exponent)
                                        {
                                            int num = 1;
                                            for (int i = 0; i < exponent; i++)
                                            {
                                                num *= number;
                                            }
                                            return num;
                                        }

                                        int main ()
                                        {
                                            exp(3, 9);
                                            std::cout << "3 to the 9th power is " << exp(3, 9) << std::endl;
                                            exp(2, 2);
                                            std::cout << "2 to the 2nd power is " << exp(2, 2) << std::endl;
                                            exp(10, 5);
                                            std::cout << "10 to the 5th power is " << exp(10, 5) << std::endl;
                                            return 0;
                                        }
