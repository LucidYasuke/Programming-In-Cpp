Iteration

❮ Previous Next ❯

Iteration involves going through something through a set or unset amount of times. In order to achieve this, we use loops. There are main types of loops in C++. For Loops and While Loops. We use them for seperate systems, sometimes both can be used for the same task but in different ways.

While Loops

A while loop loops based off of a condition. As long as a condition is met, the while loop will continue running. For instance if you have a number counter and want to print as long as a value is less than 5.
Sidenote: A do while loop always executes the codeblock once even if condition isn't intially met.

Syntax
                                    
                                        while (condition) 
                                        {
                                            // code block to be executed
                                        }
                                    
                                
                                    
                                        #include <iostream>
                                        
                                        int main ()
                                        {
                                            int bottles = 10;
                                            while (bottles > 0)
                                            {
                                                std::cout << i << " bottles of milk on the wall " << i << " bottles of milk. ";
                                                std::cout " Take one down, pass it around " << --i << " bottles of milk on the wall.\n";
                                            }
                                        }
Syntax
                                    
                                        do{
                                            // code block to be executed
                                        }
                                        while (condition)
For Loops

A for loop is similar too a while loop. But there is one big difference. A for loop takes in 3 statements. The initial condition, a stop condition, and an increment value. Once the stop condition is reached, the for loop stops.

Syntax
                                    
                                        for (statement 1; statement 2; statement 3) 
                                        {
                                            // code block to be executed
                                        }
                                    
                                
                                    
                                        #include <iostream>
                                        #include <vector>
                                        
                                        int main ()
                                        {
                                            // Iterate through a vector
                                            std::vector<std::string> colors = {"RED", "YELLOW", "BLUE", "VIOLET", "ORANGE", "GREEN"};

                                            // Statement 1 is an initialization of a number, as it is always an integral number
                                            // Statement 2 is the size of the vector, i is the index, as long as it's less than the size.
                                            // Statement 3 is the increment, we are incrementing by one
                                            std::cout << "Printing all colors\n";
                                            for (int i = 0; i < colors.size(); i++)
                                            {
                                                std::cout << "Color at index " << i << colors[i] + "\n";
                                            }

                                            std::cout << "Printing all colors backwards\n";
                                            for (int i = colors.size() - 1; i >= 0; i--)
                                            {
                                                std::cout << "Color at index " << i << colors[i] + "\n";
                                            }

                                            std::cout << "Printing every other colors\n";
                                            for (int i = 1; i < colors.size(); i += 2)
                                            {
                                                std::cout << "Color at index " << i << colors[i] + "\n";
                                            }
                                            return 0;
                                        }
