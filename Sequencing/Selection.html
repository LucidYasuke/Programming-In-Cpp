Selection

❮ Previous Next ❯

Selection is the first parts of a succesful algorithm. Algortithms uses selection to determin a different set of steps to execute based off of Boolean expressions (conditionals) or cases. There are other forms of selection but we will be going over if-else statements and switch statements

If-Else Statements

We all work on conditionals on a daily basis. If it is hot, we go outside in short. If it is cold, we go outside in a jacket. If it is a weekday, we go to school. Else, we stay home. These are simple conditionals we go through. There are more complex conditionas, such as if it is cold and rainy, we put on a jacket and bring a rain coat.
If-Else statements tells the computer how to run our program based off of conditionals given.

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {
                                        bool isCold = true;
                                        bool isRainy = false;
                                        bool isSnowy = true;
                                      
                                        // To create an if-else statement, you must always have an if; Else is optional
                                        // Bracktets are used too tell the computer what is inside the conditional
                                        // When you put a conditional or iteration inside another, it's called nesting
                                        if (isCold)
                                        {
                                          std::cout << "Today is cold, I should put on a jacket" << std::endl;
                                          // This is a nested if statement
                                          if (isRainy)
                                          {
                                            // Just a reminder you can use \n with strings
                                            std::cout << "I should also bring an umbrealla\n";
                                          }
                                          if (isSnowy)
                                          {
                                            std::cout << "Its also snowy, I should wear snow boots\n";
                                          }
                                        }
                                        else if (isRainy)
                                        {   
                                          std::cout << "Today is rainy and hot, it feels like the rainforest" << std::endl;
                                        }
                                        else
                                        {
                                          std::cout << "Today is a good day to go to the park" << std::endl;
                                        }
                                        
                                        return 0;
                                      }
                                    }
Switch Statements

Switch statements work on a case based system. Switch statements only work with integrals. The data types include integers, characters, booleans, and enums. Enums is something we will cover later, but a breif description is variables under the same category seperated by number. So for the days in the week we'd use enum, as we dont want to change them but constantly use them throughout. Switch statements are given a value, and runs it through it's cases, if it's case matches the value it will do what was programend in that area, else it will refer to its defualt.

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {

                                        char day = 'w';

                                        switch(day)
                                        {
                                            // U for sunday
                                            // Without a break, a case once done will go to the next case until it encounters a break
                                            // If we change day to 's' we will get the case under 'u'
                                            case 's':
                                            case 'u':
                                                std::cout << "It's the weekend lets have fun" << std::endl;
                                                break;
                                            // A defualt is used if the day does not match any explicit cases
                                            default:
                                                std::cout << "It's the weekday lets go to work" << std::endl;
                                                break;
                                        }
                                        return 0;
                                    }
