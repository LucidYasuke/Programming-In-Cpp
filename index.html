First Script
                            
                                // my first program in C++
                                #include <iostream>
                                using namespace std;
                                
                                int main ()
                                {
                                    cout << "Hello World!";
                                    return 0;
                                }

If you typed in the code correctly, the program should print Hello World in the console/terminal. To give some information, the first line is a comment, and is not seen as code when compiling. Include, includes libraries, iostream being built in. The next line is a namespace, when creating your own code, such as APIs, you'd want to give them all a specific name. The C++ built-in library is called the Standard, std, library. cout is a part of that library. If you didn't use namespace std, you'd have to use std::cout everytime you call cout.

                            
                                #include <iostream>
                                
                                int main ()
                                {
                                    std::cout << "Hi, My Name is John\n";
                                    std::cout << "I am 30 years old\n";
                                    return 0;
                                }

This is an example, where you see John, and the age number, change it to yours and execute the code.

If you saw your name and age be print, it was done right.
