Classes & Objects

‚ùÆ Previous

A class is a blueprint that can be edited to store values and perform methods (functions specific to that class). Say you wanted to create multiple people with different attributes. It would be a hassle to create numerous variables for each person. This is where classes come in, we create a person class which has predifined attributes and functions, and everytime we call this class, we create an object. The person is the class, the individual versions are the objects.

For this lesson, we will use header files along with cpp files. The importance of header files is to declare funtion, classes, and more. And have them be used in multiple different files without them constantly being rebuilt. With compilers, it will try and define multiple versions of the same function if they are included in multiple files.

Syntax
                                
                                    // Header File
                                    class Foo
                                    {
                                        private:
                                        protected:
                                        public:
                                            Foo();
                                            virtual ~Foo();
                                    };

                                    // Cpp File
                                    Foo::Foo()
                                    {

                                    }
                                    Foo::~Foo()
                                    {

                                    }
Header
                                
                                    #include <iostream>
                                    
                                    class Human
                                    {
                                    private:
                                        int age;
                                        std::string firstName;
                                        std::string lastName;
                                    public:
                                        Human(); // This is a default coonstructor
                                        Human(std::string firstName, std::string lastName, int age);
                                        virtual ~Human();

                                        int getAge(); // This is a method
                                        std::string getFirstName();
                                        std::String getLastName();

                                        void setAge(int age);
                                        void setFirstName(std::string name);
                                        void setLastName(std::string name);
                                    };
cpp
                                
                                    #include "Human.h"

                                    Human::Human()
                                    {
                                        this->age = 25;
                                        this->firstName = "John";
                                        this->lastName = "Doe";
                                    }

                                    Human::Human(std::string firstName, std::string lastName, int age)
                                    {
                                        this->age = age;
                                        this->firstName = firstName;
                                        this->lastName = lastName;
                                    }

                                    Human::~Human()
                                    {

                                    }

                                    int Human::getAge() // We use getters and setters if attributes are private
                                    {
                                        return this->age;
                                    }

                                    std::string Human::getFirstName()
                                    {
                                        return this->firstName;
                                    }

                                    std::string Human::getLastName()
                                    {
                                        return this->lastName;
                                    }

                                    void Human::setAge(int age)
                                    {
                                        this->age = age;
                                    }

                                    void Human::setFirstName(std::string name)
                                    {
                                        this->firstName = name;
                                    }

                                    void Human::setLastName(std::string name)
                                    {
                                        this->lastName = name;
                                    }

An attribute is a variable specific too the object. Your name for instance is an attribute.
A method is a function specific too the object.
Constructors are used to initiate an object. In other words, constructors are used to create an object.
Destructors are called once the object comes out of scope, is no longer called. It normally has nothing in it, but if pointers in your class are attributes, they should be deleted in the destructor in order to avoid memory leakage. However, if this pointer is shared by other objects in the program, it is best not too delete it if it is not the last object to hold it.
We use getters to get an attribute that is private or protected. These attributes are encapsulated and can't be accessed publicy by anything else in the program. Setters are used too change them.
