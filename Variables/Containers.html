Containers

❮ Previous Next ❯

Containers are data types that hold data. A common phrase is a list. In C++ we don't have list, but I will use the term list to expain. A list can hold items in it, in c++ a list can only hold data of a data type it was given. A list of integers can only hold integers, a list of booleans can only hold booleans.

Mutable: Capable of or subject to change or alteration.
An container type that is mutable can change its size. If you have a storage container, it has a set amount of rows and columns. You can't add too that storage container, so it is immutable. But if you had a storage container that you can add or removes rows and columns at any point, then it is mutable.

Ordered: Having elements arranged or identified according to a rule.
For common container types, Arrays and Vectors, they are ordered based off of an index. Strings are also based off an index. You've been taught when you count, start at one, don't count zero. But an index starts at zero. A container with a size of 7 items, will start at index 0 and end at index 6. We can also go backwards, if we use negative numbers, it will base it off of the last index. However, there is no such thing as -0, so we start with -1.

Arrays

An array is an ordered immutable container data type. Once a size for an array is declared, it can not be changed. Arrays are built in and do not require an include. Arrays take up less bytes than its mutable counterpart, vectors. Arrays are best used when you already have a size limit, capacity. With Arrays and Vectors, nth-dimensional containers are possible. This means that an array can hold an array inside of it, and so fourth. Below will show declaration and usage of an array. To declare an array data_type variable_name[size] or data_type variable_name[]

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {
                                        // Initialize
                                        std::string primaryColors[] = {"RED", "GREEN", "BLUE"};; 

                                        /**
                                        Alternatively
                                        Declare
                                        std::string primaryColors[3];

                                        Definition
                                        primaryColors[0] = "RED";
                                        primaryColors[1] = "GREEN";
                                        primaryColors[2] = "BLUE";
                                        **/

                                        /**
                                        Alternatively
                                        Initialize
                                        std::string primaryColors[3]  = {"RED", "GREEN", "BLUE"};
                                        **/

                                        std::cout << "Primary Color at First Index: " + primaryColors[0] << std::endl;
                                        std::cout << "Primary Color at Second Index: " + primaryColors[1] << std::endl;
                                        std::cout << "Primary Color at Third Index: " + primaryColors[2] << std::endl;

                                        // Change Element
                                        primaryColors[1] = "YELLOW";

                                        std::cout << "Primary Color at First Index: " + primaryColors[0] << std::endl;
                                        std::cout << "Primary Color at Second Index: " + primaryColors[1] << std::endl;
                                        std::cout << "Primary Color at Third Index: " + primaryColors[2] << std::endl;

                                        return 0;
                                    }

Containers and other data types may have methods built into them, some may be shown in future tutorials, but they are not being listed.

Vectors

A vector is an ordered mutable storage data type. Vectors are included in the C++ Standard Library and need too be included in the file. Vectors can also hold themselves inside create nth-dimensional vectors. Vectors holds up more memory than arrays due to them being changing, they are considered dynamic. To declare a vector std::vector<data_type> variable_name.

                                
                                    #include <iostream>
                                    #include <vector>
                                    
                                    int main ()
                                    {
                                        // Initialize
                                        std::vector<int> numbers {43, 585, 10, 495, 3, 9};

                                        /**
                                        Alternatively
                                        
                                        Method 1
                                        std::vector<int> numbers = {43, 585, 10, 495, 3, 9};

                                        Method 2
                                        std::vector<int> numbers(43, 585, 10, 495, 3, 9);
                                        **/


                                        // Add Element
                                        numbers.push_back(71);

                                        // Access Elements
                                        std::cout << "Number at Index 5: " << numbers[5] << std::endl;
                                        std::cout << "Number at Index 5: " << numbers.at(5) << std::endl;

                                        // Change Element
                                        numbers.at(-1) = numbers[0];

                                        std::cout << "Number at First Index: " << numbers[0] << std::endl;
                                        std::cout << "Number at Last Index: " << numbers.at(-1) << std::endl;

                                        // Remove Element
                                        numbers.pop_front(); // Removes first element
                                        numbers.pop_back(); // Removes last element

                                        std::cout << "Number at First Index: " << numbers[0] << std::endl;
                                        std::cout << "Number at Last Index: " << numbers.at(-1) << std::endl;

                                        std::cout << "Number at Index 5: " << numbers[5] << std::endl;
                                        numbers.erase(5) // Removes element at index 5
                                        std::cout << "Number at Index 5: " << numbers[5] << std::endl;

                                        return 0;
                                    }
