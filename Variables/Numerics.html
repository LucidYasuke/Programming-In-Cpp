Variables

Similar in mathematics, a variable is a named unit of data. For instance, the variable "people" can hold how much much there are in a room. "People" can equal 5, or it can equal 39. Ofcourse, "people" can't be equal too 9.4 as people can't be decimals. We will go over the 3 main numeric variable types int, float, and double

Integers

An integer is a numeric data type, that does not contain a decimal. In other words, int stores whole numbers without decimals. Like earlier, you'd use an integer if you wanted to store things that should only be in non-decimal numbers: people, fingers, houses, and planets. Depending on the type of integer, it wil hold less data. For this lesson, we will use the default int declaration, but it is recommended to try others.

The most common way too declare a variable is data_type variable_name;.
To define (definition) a variable, variable_name = number;.
There is a faster way too do this, called initialization, data_type variable_name = number;.
Make it note, you can not define a variable without first declaring or initializing a variable.

In the code below, we will be creating ints and printing them too the console.

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {
                                        int houses;
                                        int people = 23;
                                        
                                        // "<< std::endl" is used too go too a new line
                                        // Because houses wasn't defined, it's given a default value of zero
                                        std::cout << houses << std::endl;
                                        std::cout << people << std::endl;
    
                                        houses = 5;
    
                                        // Houses value is different
                                        std::cout << houses << std::endl;
    
                                        return 0;
                                    }
Floats

Floats, floating point numbers, are fractional numbers that contains one or more decimal digit. Floats precision, the number of decimal digits, is 7. So a float can be 3.3748205. Floats are usually used for graphics libraries due too their lower precision than doubles. A float is 4 bytes of data.

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {
                                        // Declare a float
                                        float width;

                                        // Define a float
                                        width = 10.45f;

                                        // Initialize a float
                                        float height = 720.f;
                                        
                                        // Print a float
                                        std::cout << width << std::endl;
                                        std::cout << height << std::endl;

                                        return 0;
                                    }

From above, it can be seen that when you define or initializa a float, you need to put "f" at the end; this is too seperate it from a double. When floats print however, they will contain "f" at the end. Also if a float won't print 0's if it is at the end with no number after it. 7.f will print as "7", but 7.05f will print as "7.05". Floats allow for faster computation than doubles on older systems.

Doubles

Doubles are fractional numbers that contains one or more decimal digit, similar to floats. However, doubles precision is 15 decimal digits. Allowing for greater amount of storage, or "precision" as it's called. Doubles is 8 bytes. Brief explanation, 1 byte is equal too 8 bits. So 4 bytes equal too 32 bits, and the range of value in 32-bit can be 232. 8 bytes is equal to 64 bits. So a float can take up a max of 232 bits while a double can take up 264 bits. Which is why it is called a double, being double a float.
We use the number 2, as bits can be either on or off, binary.

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {
                                        // Declare a double
                                        double width;

                                        // Define a double
                                        width = 10.45;

                                        // Initialize a double
                                        double height = 720.0;
                                        
                                        // Print a double
                                        std::cout << width << std::endl;
                                        std::cout << height << std::endl;

                                        return 0;
                                    }
Casting

Now when declaring these 3 numeric variables, you may accidentally put "1" one with no decimal place for a double. The program will automatically "cast" the type into it's data type. So 1 will become 1.0, stored in the variable. But say you wanted too forceibly cast an data type too another. You'd use the method static_cast<type>(value);. It is best practice to explicitely tell your program to change a variable type if it needs to be changed.

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {
                                        double avgTimeD = 3.45454545;
                                        float avgTimeF = avgTimeD;
                                        int avgTimeI = avgTimeD;
                                        
                                        // On compilers that allow, shows all 8 decimal digits for avgTimeD
                                        std::cout << avgTimeD << std::endl;
                                        
                                        // On compilers that allow, shows too 4 and drop the last 5, does not round
                                        std::cout << avgTimeF << std::endl;

                                        // Note : IDEs when compiling may choose too round and drop digits

                                        // Will drop all decimal digits, it does not round
                                        std::cout << avgTimeI << std::endl;

                                        // Casting
                                        int avgPeople = static_cast<int>(121 / 3.0);
                                        std::cout << "The average amount of people is " << avgPeople << std::endl;

                                        return 0;
                                    }

When operating between 2 different numeric data types, such as adding an integer and double, the return value will have a type with the higher byte. Subtracting a float and double will return a double value. Multiplying an integer and float will return a double
