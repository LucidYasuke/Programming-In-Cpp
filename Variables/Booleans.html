Booleans

❮ Previous Next ❯

Booleans are true and false values. They're represented in one byte and are either on or off. Computers working in binary, 0 is off and 1 is on. So a true value is one bit, while a false value is 0 bits. A boolean default value is false. There aren't much too booleans on their own beside representing true or false. Alternatively, you can use 0 or 1 to represent a boolean. Booleans are normally used with conditionals and operators to run, which will be seen later. There are multiple ways too print a boolean, we will be showing one way.

                                
                                    #include <iostream>
                                    
                                    int main ()
                                    {
                                        std::string day = "wednesday";
                                        bool isWeekday = true;
                                        bool isWeekend = false;

                                        std::cout << "Today is " + day << std::endl; // Output : Today is wednesday
                                        std::cout << "Is weekday: " + std::string(isWeekday ? "true" : "false") << std::endl; // Output : Is weekend: true
                                        std::cout << "Is weekend: " + std::string(isWeekend ? "true" : "false") << std::endl; // Output : Is weekend: false

                                        return 0;
                                    }
