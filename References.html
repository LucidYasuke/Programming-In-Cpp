References

❮ Previous Next ❯

A reference is the RAM (memory) Address where a variable is stored. References can instead of sending a copy of the value stored in a variable, telling a program where the variable is stored and to use it from here. In a later lesson, you will learn about encapsulation. With encapsulation, some variables may be private too the program, and can only be accesses from ourside using their reference. Or you may want too share a variable between different functions or scopes. But what if you wanted to store that reference somewhere. A variable holding the reference to another variable is called a pointer.

To reference a variable we use the "&" symbol, &variable.
To create a pointer we use the "*" symbol and must use it after the data_type, data_type*;.
To dereference, or get the value stored at a pointer, we use the "*" symbol *pointer;

                            
                                #include <iostream>

                                int main ()
                                {
                                    int people = 7;
                                    int* peoplePointer = &people;

                                    std::cout << "Number of people: " << people << std::endl;
                                    std::cout << "Value is stored at: " << peoplePointer << std::endl; 
                                    return 0;
                                }
                            
                        
                            
                                #include <iostream>

                                // The function changes the value giving too its exponent at the end
                                void exp(int* num, int exponent)
                                {
                                  int value = 1;
                                  for (int i = 0; i < exponent; i++)
                                  {
                                    value *= *num;
                                  }
                                  *num = value;
                                }
                                
                                int main ()
                                {
                                  int people = 7;
                                  std::cout << "The old value of people is " << people << std::endl;
                                  exp(&people, 2);
                                  std::cout << "The new value of people is " << people << std::endl;
                                  return 0;
                                }
